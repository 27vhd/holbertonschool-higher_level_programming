# Python More Data Structures

This repository contains examples and exercises for working with advanced Python data structures.

## Contents

- **Lists** - Comprehensive list operations and methods
- **Dictionaries** - Key-value pair manipulation and iteration
- **Sets** - Unique element collections and set operations
- **Tuples** - Immutable sequences and unpacking
- **Collections** - Counter, defaultdict, and namedtuple examples

## Getting Started

1. Clone or navigate to this repository
2. Open Python files in your preferred editor
3. Run examples with `python filename.py`

## Learning Resources

- [Python Official Documentation](https://docs.python.org/)
- Built-in data structure methods and properties
- Real-world use cases for each structure
