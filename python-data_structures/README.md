# python-data_structures

Lightweight collection of Python exercises and examples focused on core data structures (lists, tuples, sets, dictionaries, stacks, queues, linked lists) for learning and practice.

## Requirements
- Python 3.8+
- (Optional) pytest or unittest for running tests

## Installation
1. Clone the repository:
    git clone <repo-url>
2. Enter the project directory:
    cd python-data_structures

## Usage
- Each task is a standalone `.py` file or module. Run examples with:
  python3 path/to/script.py

## Running tests
- With unittest:
  python3 -m unittest discover
- With pytest (if available):
  pytest

## Project layout
- tasks/ or exercises/ — individual exercises and solutions
- tests/ — unit tests for exercises
- examples/ — usage examples and demos

## Style
- Follow PEP 8
- Write small, well-documented functions with clear names
- Keep functions focused and testable
