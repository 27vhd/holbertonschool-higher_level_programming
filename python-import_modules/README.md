
# Python - Import Modules

## Description
This project covers Python import modules, including how to import functions, classes, and variables from other modules and packages.

## Learning Objectives
- Understand how to import modules in Python
- Use `import` and `from ... import` statements
- Explore the `__name__` variable
- Work with modules and packages
- Use the `dir()` function to inspect modules

## Topics Covered
- Module basics and `sys.modules`
- Importing specific functions and classes
- Aliasing imports with `as`
- Wildcard imports (`from module import *`)
- Creating and structuring packages
- Built-in modules

## Resources
- [Python Official Documentation - Modules](https://docs.python.org/3/tutorial/modules.html)
- [Python Import System](https://docs.python.org/3/reference/import.html)

## Files
- `*.py` - Python scripts demonstrating import concepts

## Usage
```python
# Import entire module
import module_name

# Import specific function
from module_name import function_name

# Import with alias
import module_name as alias
```
